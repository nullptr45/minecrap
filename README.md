# Minecrap

A C++20 Minecraft clone using OpenGL 3.3

## Features

- OpenGL 3.3 renderer
- JSON-defined block registry
- Data-driven architecture
- ImGui debug UI

---

## Dependencies

This project uses git submodules for third-party libraries.

### Libraries

- SDL2 – Windowing & input
- GLAD – OpenGL loader
- glm – Math library
- nlohmann/json – JSON parsing
- Tracy – CPU & GPU profiler
- stb_image – Image loading
- Dear ImGui – Debug UI

---

## Build Instructions

### Requirements
- Git (obviously)
- CMake 3.16+
- C++20 compiler
- OpenGL 3.3+ capable GPU

### Cloning
Clone the repository with submodules:

```bash
git clone --recursive https://github.com/nullptr45/minecrap.git
cd minecrap
```
